Comment ajuster les coordonnées

Système de coordonnées PDF:

  • X = 0 à gauche de la page, augmente vers la droite
  • Y = 0 en bas de la page, augmente vers le haut
  • Page A4 = 595 × 842 points
  • Si le texte est trop à gauche → augmenter X
  • Si le texte est trop haut → diminuer Y
  • Si le texte est trop bas → augmenter Y
